HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS stands for Cascading Style Sheets
  • CSS allows us to give our content style
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can style a webpage using inline CSS, internal CSS style sheet, and the most common, external CSS style sheet.

Git

  • Git is a version control system that lets us manage each version of our project
  • Git commands are used to save and track different versions of our code on our own computer
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript allows users to interact with a website
